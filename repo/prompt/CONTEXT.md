## Business Model Details

**Niche distribution:** Roofing 80–90% of clients, HVAC ~10%, Gutters ~5%.

**Client ICP (Ideal Customer Profile):**
- $2.5M+ annual revenue
- 3+ years in business
- Growth-minded owner
- $3,000+/month ad spend capacity

**Revenue model per client:**
- Setup fee (one-time, paid upfront)
- Retainer: ~$4,000/month base + 10% of ad spend
- 10% = scalability fee: more spend = more leads = more VA time = more management

**The 28-day cycle:**
- Clock starts when ads go live (not when contract is signed)
- Pod manager must hit 80% of agreed booking target to bill
- If target not hit, cycle extends at no charge until target is reached
- "Speed to Cashflow" target: 60 days or less from signed contract to first payment (company average is 75–80 days — this is the gap to close)

**The 80% rule:** If client agrees to 15 bookings/month, pod manager must deliver minimum 12 (80%) before collecting the retainer.

## The Funnel (Know This Cold)

```
Meta Ads → Landing Page (Survey) → GHL CRM → VA Phone Call → Booked Appointment
                                                                      ↓
                                                              Layer 1 Metric
```

**Layer 1 metrics (what you check first):**
- Booked appointments vs target
- Cost per booking
- Health status (green / yellow / orange / red)

**Layer 2 metrics (only check if Layer 1 is failing):**
- CPC (cost per link click)
- CTR (link click-through rate)
- Survey conversion rate
- CPL (cost per lead — leads who filled out survey)
- Out-of-service-area lead rate

**Rule:** If Layer 1 is healthy → do NOT dig into Layer 2. This protects time across a 25+ account book.

## Creative Standards (Creative Mastery: Ad Set 101)

**3 Photo Types:**
- **Trust:** Happy person (solo, team, roofer+homeowner, happy homeowner). Always involves a face.
- **Service:** Completed roof (ideal) or work in progress (less ideal). No people required.
- **Brand:** Company vehicle, branded clothes, lawn signs.

**3 Ad Formats:**
- **Single:** 1 full-frame photo. Must be Trust or Service only.
- **Two-Fold:** 2 stacked photos. Top = Trust. Bottom = Trust OR Service OR Brand.
- **Tri-Fold:** 3 photos in L-shape. Top-left = Trust. Top-right = anything. Bottom = anything.

**Ad Set Standard:** 5 Singles + 5 Two-Folds + 5 Tri-Folds = **15 ads per set minimum**. Videos are additive (on top of the 15 if requested).

**Once a creative is used in a set — never reuse it in another set.**

**AI Image Rules:**
- Logos must be perfect (no blurring, distortion)
- No jumbled text
- No "twins" (same person appearing twice in one image)
- Don't spam the logo in every image
- If client reps' photos exist, use their likeness — no generic faces for owner/sales roles

## The Post-Andromeda Creative Refresh Protocol

Meta's Andromeda update concentrates spend on top 3–4 ads. Standard creative fatigue process:

1. Sort ads by reach in ad set — observe top 3-4 getting 90%+ of spend
2. Check 7-day vs 30-day cost per link click — has it increased significantly?
3. If Layer 1 metrics are failing AND creative fatigue signals present:
   - Go to ad set level
   - Turn off the top-reach ads
   - Duplicate the ad set (no other changes)
   - Turn on new ad set, turn off old one
   - Wait 48 hours — 50% chance it resets performance
4. If duplicate attempt fails → full creative refresh with new images (don't reuse any past ads)
5. If client has new content → build a fresh campaign from scratch

## Common First-Month Mistakes (Based on Pod 1)

1. **Calculating CPL instead of cost per booking** — divide spend by appointments, not leads
2. **Ignoring context** — making changes based on 7-day windows that include holidays
3. **Reusing creatives** — same photo 8x in different ads is not variety
4. **Not setting up CAPI** — running ads with zero pixel conversion data
5. **Campaign names missing "B2C"** — dashboard doesn't track them
6. **Forgetting to turn off Advantage+** — Meta adds it automatically on duplicates
7. **Waiting on clients instead of calling** — ghosting clients get calls, not Slack messages
8. **Looking at Layer 2 when Layer 1 is fine** — wastes time that should go to flagged accounts
9. **Ignoring open leads** — white leads in the logbook that VAs haven't booked yet are lost revenue
10. **Not boosting new Facebook pages** — launching with 0 followers hurts ad performance
