# v1 Reference

This directory contains the v1 onboarding HTML file for reference when building v2.

## Files

- `ci_onboarding_v1.html` — The complete v1 deliverable (~215KB). Open in any browser.
- See `../docs/v1-feature-analysis.md` for a detailed breakdown of what's in v1, what works, and what v2 should improve.

## How to Use This Reference

1. Open `ci_onboarding_v1.html` in a browser to see the full v1 experience
2. Read `../docs/v1-feature-analysis.md` for a code-level breakdown
3. The goal is to beat v1 on: quiz retention, day completion rate, return engagement, and admin visibility
4. v1 is the baseline. v2 should win decisively on all metrics.
